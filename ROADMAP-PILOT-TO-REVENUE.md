# redis-stig-audit — Pilot-to-Revenue Roadmap (Draft / Saved for Later)

Status: parked-draft
Owner: Keith

## Goal
Turn Redis container audit content into a pilot-ready assessment tool for regulated environments that need recurring Redis security evidence.

## Near-term product direction
- Draft benchmark document
- Working runtime audit CLI
- Enterprise-friendly outputs (JSON, SARIF, Wiz, SCC)
- Assessment/evidence bundle for annual reviews

## Potential use cases
- FedRAMP annual security assessments
- Redis hardening reviews in cloud/container environments
- CI/CD guardrails for managed self-hosted Redis estates

## Early validation questions
1. Is there enough market gap in Redis-specific container audit tooling?
2. Which target matters more first: FedRAMP contractors, MSSPs, or internal enterprise security teams?
3. How much of the value comes from benchmark content vs automated runtime checks?
